# **CEFAEMPRESA**

CEFAEMPRESA es software creado con el proposito de gestionar la información que manejan en **SENA Empresa**, pero tambien encontramos documentación relaciónada a las herramientas que puedes utilizar como programador, y como administrar lo que sea un VPS (virtual private server).


Con el proposito de que sea un proyecto a largo plazo, facil de mantener y moderno se implementaon ciertas tecnologias, y configuraciónes especificas, se encontra documentación de como manejar todos estos elementos.

--- 

## Tecnologias

### Manejo de verciones y comunicación entre desarolladores.

* **Githab** :  Es una plataforma online en la cual se puede subir el proyecto y separarlo en verrciones, le permite a los programadores pasarse codigo y manejar la integración de nuevos modulos modificaciones con un control minusioso.

* **Git** : es una tecnologia que nos permite desde nuestros computadores ejecutar comandos para manejar verciones del software de manera local o en la nube.

> tomar en cuenta que se utiliza lo que seria Git Flow para manejar las verciones del software.
